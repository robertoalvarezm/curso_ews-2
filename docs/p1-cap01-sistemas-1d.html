Sistemas dinámicos 1-D y estabilidad

Objetivos de aprendizaje

  • Interpretar una EDO autónoma \(\dot{x}=f_\mu(x)\) como un flujo sobre la recta.
  • Localizar puntos fijos y decidir su estabilidad de forma intuitiva (signo del flujo) y formal (linealización).
  • Reconocer el papel del parámetro de control \(\mu\) como antesala de las bifurcaciones.

1 La recta como espacio de fases

Muchos sistemas biológicos cuya variable de interés es función del tiempo numérica, por ejemplo el tamaño de una población, la densidad de un nutriente, la fracción de individuos infectados, la concetración química, pueden describirse con una ecuación diferencial ordinaria autónoma:

\[ \dot{x} \;=\; \frac{dx}{dt} \;=\; f_\mu(x). \tag{1}\]

Aquí

  • \(x(t)\) es el función del tiempo que describe el estado del sistema,
  • \(f\) es una función suave que dicta su tasa de cambio ( el mdoelo del sistema), y
  • \(\mu\) es un parámetro de control (o, en general, un vector de parámetros) que el ambiente o un experimento detrminan: una tasa de dilución, un nivel de luz, una carga de nutrientes.

Que sea autónoma significa que \(f\) no depende explícitamente del tiempo: la regla de evolución es la misma hoy que mañana. Eso nos permite pensar geométricamente. Imagina una partícula sobre la recta cuya velocidad en cada punto \(x\) es \(f_\mu(x)\): donde \(f>0\) se mueve a la derecha, donde \(f<0\) a la izquierda.

2 Puntos fijos y estabilidad

Un punto fijo (o equilibrio) \(x^*\) cumple \[ f_\mu(x^*) = 0, \tag{2}\] es decir, el sistema no cambia si parte exactamente de ahí. La pregunta biológica relevante es: si lo perturbamos un poco, ¿regresa o se aleja?

NotaDefinición formal (estabilidad lineal)

Sea \(x^*\) un punto fijo de Ecuación 1. Linealizando con \(x = x^* + \eta\) y \(\eta\) pequeño, \(\dot{\eta} \approx f'_\mu(x^*)\,\eta\), cuya solución es \(\eta(t)=\eta_0\,e^{f'(x^*)\,t}\). Por tanto:

  • Si \(f'_\mu(x^*) < 0\), las perturbaciones decaen ⇒ \(x^*\) es estable (atractor).
  • Si \(f'_\mu(x^*) > 0\), las perturbaciones crecen ⇒ \(x^*\) es inestable (repulsor).
  • Si \(f'_\mu(x^*) = 0\), el criterio lineal no decide: hay que mirar los términos de orden superior (esto es justo lo que ocurre en una bifurcación).

El inverso de \(|f'(x^*)|\) es, además, una escala de tiempo de recuperación: cuanto más cerca de cero esté la pendiente, más lento regresa el sistema tras una perturbación. Guarda esta idea: es la semilla del enlentecimiento crítico de la Parte III.

3 Ejemplo guía: crecimiento logístico

Tomemos el clásico modelo logístico de una población, \[ \dot{x} = r\,x\left(1 - \frac{x}{K}\right), \tag{3}\] con tasa intrínseca \(r>0\) y capacidad de carga \(K\). Tiene dos puntos fijos: \(x^*=0\) y \(x^*=K\).

Mueve la condición inicial y observa hacia dónde converge la trayectoria:

viewof x0 = Inputs.range([0.01, 2], {value: 0.1, step: 0.01, label: "x₀ (condición inicial)"})
traj = {
  const r = 1, K = 1, dt = 0.05, T = 240;
  let x = x0; const pts = [];
  for (let i = 0; i <= T; i++) { pts.push({t: i*dt, x}); x = x + dt*(r*x*(1 - x/K)); }
  return pts;
}
Plot.plot({
  height: 280, marginLeft: 48,
  x: {label: "tiempo t"},
  y: {label: "x(t)", domain: [0, 2]},
  marks: [
    Plot.ruleY([1], {stroke: "crimson", strokeDasharray: "4 4"}),
    Plot.ruleY([0], {stroke: "gray", strokeDasharray: "2 4"}),
    Plot.line(traj, {x: "t", y: "x", stroke: "steelblue", strokeWidth: 2})
  ]
})

Toda condición inicial positiva converge a \(x^*=K\) (línea roja): es el equilibrio estable. El \(x^*=0\) es inestable.

4 Ejercicios

TipEjercicio 1.1 — Analítico

Para \(\dot{x} = x - x^3\), encuentra todos los puntos fijos y clasifícalos usando \(f'(x^*)\).

NotaSolución

\(f(x)=x-x^3 = x(1-x^2)=0 \Rightarrow x^*=0,\,\pm 1\). Con \(f'(x)=1-3x^2\): \(f'(0)=1>0\) (inestable); \(f'(\pm 1)=-2<0\) (estables). Es un sistema biestable: dos atractores separados por un repulsor.

TipEjercicio 1.2 — Computacional

En la celda de Python, cambia r a un valor negativo. ¿Qué punto fijo se vuelve estable ahora? Justifica con el signo de \(f'(x^*)\).

NotaSolución

Con \(r<0\), \(f'(0)=r<0\) (estable) y \(f'(K)=-r>0\) (inestable): se intercambian los papeles. Toda población decae a \(0\). Esto anticipa la bifurcación transcrítica del capítulo siguiente.

TipReto — Flujo sobre la recta

Esboza el retrato de fases de \(\dot{x}=\sin x\). ¿Cuántos puntos fijos hay y cómo se alternan en estabilidad?

NotaPista / solución

Puntos fijos en \(x^*=n\pi\). Como \(f'(x)=\cos x\), los \(\cos(n\pi)=(-1)^n\) alternan signo: los \(n\) pares son inestables y los impares estables (o viceversa según el signo). El flujo empuja cada trayectoria hacia el equilibrio estable más cercano.
